# 🎵 Spotify UI Clone (Self-Built)

This project is a **self-built Spotify-inspired UI clone** created using HTML and CSS. The design is developed completely from scratch without copying or using any existing repository.

## 🚀 Features

* 🎧 Modern music player UI inspired by Spotify
* 📂 Sidebar navigation (Home, Search, Library)
* 🎶 Playlist and album sections
* ▶️ Music player bar UI (play, pause, controls design)
* 📱 Responsive layout for different screen sizes
* 🎨 Clean and visually appealing design

## 🛠️ Tech Stack

* HTML5
* CSS3

## 💡 About the Project

This project is focused on improving front-end skills by recreating a real-world music streaming interface. It helps in understanding layout design, flexbox/grid, and responsive styling techniques.

## 🔥 What Makes It Special

* ✅ Fully designed and coded by me
* ✅ No frameworks or libraries used
* ✅ Clean UI inspired by real-world application
